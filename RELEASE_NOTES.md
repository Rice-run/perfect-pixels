# Perfect Pixels 0.1.2 中文更新日志

这一版重点补齐了像素逐帧编辑时最常用的导出和画布操作体验。

## 新增

- 新增导出预设：
  - 当前帧 PNG
  - 横向 Sprite Sheet
  - 纵向 Sprite Sheet
  - GIF 预览
  - 每帧单独 PNG zip
- 新增右键吸色。
- 新增鼠标中键拖动画布。
- 新增鼠标滚轮缩放，并围绕鼠标位置缩放。
- 新增工具快捷键自定义，可在左侧面板修改并自动保存。
- 新增图标化工具栏，减少文字按钮占用。
- 新增状态栏，显示当前坐标、颜色、帧号、图层和缩放比例。
- 新增网格规格选择，不同画布尺寸可以使用不同密度的检查网格。
- README 新增 GIF 演示。

## 改进

- 导出按钮改为「预设 + 执行」模式，减少误点不同导出格式的成本。
- 工具按钮悬停会显示工具名称和当前快捷键。
- GIF 导出补齐类型声明，网页构建可以正常通过。

## 验证

- 已通过 `npm run build`。
